
# ConsultAI NZ Vision

## Project Vision
AI-powered medical consultation platform designed specifically for New Zealand healthcare professionals, focusing on:
- Streamlining consultation process
- Real-time transcription
- Intelligent note generation
- Integrated NZ health resources

## Target Users
Primary Users:
- General Practitioners in New Zealand
- Medical professionals requiring consultation documentation
- Healthcare providers needing efficient note-taking solutions

## Development Focus
- New Zealand healthcare context
- GP workflow optimization
- Medical documentation needs
