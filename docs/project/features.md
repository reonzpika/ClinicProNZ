
# ConsultAI NZ Core Features

## 1. Live Transcription
- Real-time transcription using Deepgram
- New Zealand English and medical terminology support
- Automatic speaker diarization (doctor vs patient)
- Optimized for medical consultation context

## 2. Intelligent Note Generation
- AI-powered consultation note generation
- Template-based structure
- Context-aware medical terminology
- Integration with transcription system

## 3. Template Prompt System
- Structured template management
- Section-specific prompts for each template
- Support for basic SOAP and multi-problem SOAP formats
- User-defined custom templates
- Template validation and error handling

## 4. User Interface
- Clean, intuitive design
- Medical workflow optimization
- Responsive layout
- Accessibility compliance
- Healthcare environment considerations
