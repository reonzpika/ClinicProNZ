# Template System Guidelines

## Template Purpose
- Define structure for AI-generated notes
- NOT forms for manual filling
- Guide ChatGPT prompt creation
- Structure consultation analysis

## Template Workflow
1. GP selects appropriate template
2. Template structure creates ChatGPT prompt
3. ChatGPT analyzes live transcription
4. AI generates structured note following template format

## Template Types

### 1. Multi-problem SOAP Template
- Covers ~80% of GP consultations
- Creates SOAP format for each problem
- Standard structure for common consultations

### 2. Specialized Templates
- Driver's license medical
- 6-week baby check-up
- Initial medical enrollment
- Other common consultation types

### 3. Custom Templates
- Created by GPs
- Private to creator
- Follow standard structure

## Template Access
- Default templates: Available to all users
- Custom templates: Only visible to creator
- Template management system
- Version control for templates
