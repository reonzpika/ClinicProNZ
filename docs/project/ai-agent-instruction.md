# AI Agent Instructions

## Understanding Project Context

### 1. Project Overview
- ConsultAI NZ is an AI-powered medical consultation platform
- Target users are New Zealand healthcare professionals
- Focus on MVP features: transcription, templates, note generation
- Built with Next.js 14, TypeScript, and modern web technologies

### 2. Key Requirements
- Real-time transcription using Deepgram
- Template-based note generation
- Simple template management system
- Clean, intuitive UI for medical workflows

## Handling Non-Technical Requests

### 1. Feature Requests
- Understand the medical context
- Consider GP workflow implications
- Evaluate against MVP scope
- Suggest phased implementation if needed
- Assess impact on existing features

### 2. UI/UX Feedback
- Focus on medical workflow efficiency
- Consider GP user experience
- Ensure intuitive navigation
- Support quick actions
