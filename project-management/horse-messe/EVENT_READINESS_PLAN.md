# Event Readiness Plan (end of Jan)

## Goal
Be fully ready to sell at Horse Messe by **31 Jan 2026** with minimal stress on the day.

## Assumptions (defaults unless changed)
- Website is **Option A (landing page)**, not full e-commerce.
- Payments are **Square + cash** (or PayPay + cash if already preferred).
- Primary customer channel is **Instagram**, with Linktree as hub.

## Deliverables checklist

### A) Products and pricing
- [ ] Final SKU list (name, colour/pattern, size, price)
- [ ] Inventory counts per SKU
- [ ] Care instructions (wash, handling)
- [ ] Packaging: bags, tissue, stickers, thank-you cards
- [ ] Price cards printed (JPY, simple, readable)

### B) Brand and customer capture
- [ ] Instagram profile set up (bio, highlights, pinned posts)
- [ ] Content runway planned (high level):
  - [ ] 12 feed posts ready by event day
  - [ ] 3–5 stories/week in the lead-up; daily stories in the final week
- [ ] Linktree links populated (Instagram, website, event info, contact)
- [ ] QR codes printed:
  - [ ] “Follow Instagram”
  - [ ] “View catalogue / website”
  - [ ] Optional: “Custom order enquiry”
- [ ] Simple DM script for enquiries

### C) Website (Option A landing page)
Minimum sections:
- [ ] Hero: Cloud Nine x 久留米絣, what makes it special
- [ ] Product gallery (8–12 images)
- [ ] Price list or price range
- [ ] Event details (where, when, booth name if known)
- [ ] How to buy: “At the event”; “Custom orders by DM/contact”
- [ ] Contact form or email/LINE

### D) Payments and receipts
- [ ] Payment provider selected and set up (KYC complete)
- [ ] Test transaction completed (small test charge and refund if needed)
- [ ] Decide receipts:
  - [ ] Printed receipt book OR digital receipt via payment provider OR hand-written simple receipts
- [ ] Cash float prepared (small notes and coins)

### E) Booth operations
- [ ] Packing list (boxes labelled)
- [ ] Booth layout plan (table, display stands, signage)
- [ ] Setup checklist (arrival time, power needs, tape/scissors, extension cord)
- [ ] Inventory sheet for day-of (start count, sold, end count)
- [ ] Sales script (30 seconds): story, materials, why special, how to care
- [ ] Contingency plan (no internet, card terminal issues, rain, low stock)

## Timeline (suggested)

### 16–19 Jan
- Decide website scope and payment choice.
- Create SKU list and provisional pricing.
- Draft website copy and gather minimum photos.

### 20–26 Jan
- Publish website MVP and connect from Linktree.
- Print QR signage and price cards.
- Complete payments setup and run a test transaction.

### 27–31 Jan
- Final inventory count and pack.
- Post final Instagram runway content.
- Rehearse event-day setup and payment flow.

## Open questions to resolve (record answers here)
- Website scope: Option A or B?
- Payment provider: Square or PayPay (or other)?
- Will you accept custom orders after the event (shipping, lead time)?
